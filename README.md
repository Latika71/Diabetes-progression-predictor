# 🩺 Diabetes Disease Progression Predictor

A Machine Learning web app that predicts diabetes disease progression score based on patient health features.

## 🔗 Live Demo
[👉 Click here to open app](https://diabetes-progression-predictor-kzt8bjd4usmgjpeyjjavs4.streamlit.app/)

## 🛠️ Tech Stack
- Python
- Scikit-learn — Decision Tree Regressor
- Streamlit
- Google Colab

## 📊 Input Features
| Feature | Description | Range |
|---------|-------------|-------|
| Age | Patient age | 19–79 years |
| Sex | Gender | Male / Female |
| BMI | Body Mass Index | 18–42 |
| BP | Blood Pressure | 62–133 mmHg |
| S1 | Total Cholesterol | 97–301 |
| S2 | LDL | 41–242 |
| S3 | HDL | 22–99 |
| S4 | TCH Ratio | 2–9 |
| S5 | Triglycerides (log) | 3.3–6.1 |
| S6 | Blood Sugar | 58–124 |

## 📈 Output
Predicted disease progression score (25–346).
- 🟢 Below 100 — Low progression
- 🟡 100–200 — Moderate progression
- 🔴 Above 200 — High progression
